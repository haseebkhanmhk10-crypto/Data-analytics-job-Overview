# 📊 Data Analytics Portfolio – Salary & Job Market Insights

This repository contains two end-to-end data analytics projects focused on **salary trends, job demand, and skill-based insights in the data industry**. The work demonstrates practical capability in **data cleaning, transformation, analysis, and dashboarding** using real-world datasets.

# 🔹 Project 1: Salary & Skills Analysis

## 📌 Introduction

This project analyzes the relationship between **job roles, salaries, and required skills** in the data domain. It focuses on identifying **what actually drives compensation** and how skill demand varies across roles.

---

## 🛠️ Skills & Tools Used

* Microsoft Excel (Advanced)
* Data Cleaning & Transformation
* Pivot Tables & Aggregation
* Data Modeling
* DAX
* Power Pivot
* Power Query 
* Exploratory Data Analysis (EDA)
* Analytical Thinking (correlation, comparison, trend analysis)

---

## ❗ Problem Statement

Most job seekers blindly learn skills without understanding:

* Which skills actually increase salary
* How many skills are expected per role
* Differences between US vs non-US salary structures

This project eliminates guesswork by quantifying these relationships.

---

## ❓ Key Questions Answered

* Which job roles offer the highest median salaries?
* Does having more skills actually increase salary?
* Which skills are most associated with high-paying jobs?
* How do salaries differ between US and non-US markets?
* What is the likelihood of specific skills appearing in job postings?

---

## 📈 Results & Insights

### 1. Salary vs Skills Tradeoff

* Senior roles (e.g., Data Scientist, Data Engineer) dominate top salary brackets
* More skills ≠ always higher salary
  → Some roles demand **depth over breadth**

### 2. Geographic Salary Gap

* US salaries significantly outperform non-US salaries
* Some roles show **almost double compensation in US markets**

### 3. Skill Value Analysis

* High-value skills include:

  * Python
  * Tableau
  * Oracle
* Skill likelihood reveals **market demand vs rarity imbalance**

### 4. Job-Skill Mapping

* Data Engineering roles heavily require:

  * Kafka
  * Hadoop
  * NoSQL
    → Indicates infrastructure-heavy skill demand

# 🔹 Project 2: Data Science Salary Dashboard

## 📌 Introduction

This project is an **interactive Excel dashboard** built to analyze **global data job salaries, job distribution, and hiring platforms**. It transforms raw job posting data into a decision-making tool.

---

## 🛠️ Skills & Tools Used

* Excel Dashboarding
* Data Visualization
* Data Cleaning
* Data Validation Techniques
* Interactive Filters (Slicers)
* KPI Design
* Data Structuring

---

## ❗ Problem Statement

Raw job data is messy and useless without structure.
Job seekers and analysts need:

* Clear salary benchmarks
* Job distribution insights
* Platform-level hiring trends

This dashboard solves that by making the data actionable.

---

## ❓ Key Questions Answered

* What is the median salary by job role?
* Which countries offer better salaries?
* How does job type (full-time, contract, etc.) affect salary?
* Which platforms post the most jobs?
* What roles dominate the job market?

---

## 📈 Results & Insights

### 1. Role-Based Salary Insights

* Senior Data Scientist → highest median (~155K)
* Data Scientist & Engineer roles consistently high-paying
* Data Analyst roles lower but more abundant

### 2. Job Distribution

* Data Analyst roles dominate volume
* Senior roles are fewer but higher paying

### 3. Country-Level Trends

* Significant variation in salaries across countries
* Developed markets dominate high salary ranges

### 4. Job Type Impact

* Full-time roles offer the highest stability and pay
* Contract roles show variability but competitive pay

### 5. Platform Analysis

* Major platforms:

  * Indeed
  * LinkedIn
  * ZipRecruiter
* Hiring volume differs significantly by platform
